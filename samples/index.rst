.. _samples:

Sample programs written for geckOS
==================================

The following is a list of programs written for geckOS in order to showcase
geckOS's main features and device abstraction benefits. They are meant to be
run on a variety of programmable devices exposing a certain set of supervisor
calls (see :ref:`supported_devices`).

Most applications are written in C, and each contains a short description and
instructions on how to build them for the target device.

..
   Order these alphabetically.
